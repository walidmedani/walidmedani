## 🚀 Projects

### [Federal Aviation Administration Registry](https://github.com/walidmedani/FAA-Registry)
A production-grade ELT pipeline that ingests the FAA Aircraft Registry (~311k active aircraft records) into BigQuery. The system is engineered to handle government data at scale, moving from raw unstructured files to a refined, AI-ready semantic gold layer.
* **Tools:** Apache Airflow, dbt, duckDB, PySpark, BigQuery, Great Expectations, Docker, SQL.

---

### [Olist E-Commerce ELT Pipeline](https://github.com/walidmedani/olist-elt-pipeline)
An automated ELT pipeline processing 100k+ records of Brazilian e-commerce data. This project orchestrates a multi-stage ingestion and transformation engine, utilizing a containerized environment to manage complex relational dependencies. The transformation layer is engineered to map raw entities into a unified semantic model, ensuring data consistency and readiness for business analytics.
* **Tools:** Apache Airflow, Apache Spark, Google Cloud Platform, Docker, Terraform, DBT, Python, PostgreSQL.

---

### [Airflow Book-Data ETL](https://github.com/walidmedani/airflow-book-etl)
A production-grade ETL system designed for automated web data collection and structured storage. It features custom Airflow operators that handle web-scraping, data validation, and error-handling at scale. The project focuses on transforming unstructured metadata into high-fidelity, strictly typed datasets suitable for grounding RAG systems and LLM applications.
* **Tools:** Apache Airflow, Python (BeautifulSoup/Requests), MySQL, XComs, Docker.

---

### [SQL Web Analytics & Behavioral Mapping](https://github.com/walidmedani/SQL-Web_Analytics)
Advanced transformation and modeling of clickstream event data. This project leverages complex SQL logic to reconstruct user sessions and conversion funnels from raw event logs. By defining precise behavioral semantics and optimizing query performance through CTEs and window functions, it provides the foundational logic required for predictive modeling and automated user clustering.
* **Tools:** Advanced SQL (PostgreSQL), Data Modeling, Analytical Engineering.
